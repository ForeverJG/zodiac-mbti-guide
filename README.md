# 🌙 星座人格聊天指南

用星座和 MBTI，更好地理解她。一个纯静态的聊天指南工具，帮你在和不同星座/MBTI类型的人相处时找到最合适的聊天方式。

## 功能

- **🌟 星座查询** — 12星座完整档案：性格特点、专属开场白（可一键复制）、相处建议、雷区提醒、沟通心法
- **🧠 MBTI查询** — 16型人格完整档案：思维模式、聊天心法、相处之道，按NT/NF/SJ/SP四组分类
- **🔗 配对匹配** — 星座×星座 / MBTI×MBTI 匹配度分析（5级评分 + 详细说明）
- **💬 聊天心法** — 8条核心沟通原则、四元素一句口诀、MBTI四组快速参考、万能话术模板
- **📝 我的笔记** — 记录个人与不同星座/MBTI的相处经验，localStorage本地存储，支持编辑和删除

## 使用方式

1. 双击 `index.html` 在浏览器中打开
2. 无需安装任何依赖，无需联网
3. 手机和电脑均可使用（响应式设计）

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- 零外部依赖，纯 Emoji 图标
- localStorage 本地持久化
- 响应式设计，移动优先（最大宽度 768px）
